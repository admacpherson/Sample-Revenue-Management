# Sample-Revenue-Management
Simulating a revenue management case study based on publicly available data

## Structure
* `data`
  * `scraper.py`: BeautifulSoup scraper that reads most current data from [DOT website](https://www.transtats.bts.gov/AverageFare/) and exports it to table.csv 
  * `table.csv`: CSV-formatted data from DOT
  * `2025Q2.csv`: Static table with 2025 Q2 data 
* `.gitignore`
* `README.md`
